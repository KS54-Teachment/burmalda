 ## PASSWORD CHECKER


 ## Password checker - Программа, которая проверяет надежность вашего пароля по нескольким критериям, ставит оценку и имеет список ваших введенных паролей.

 ## Инструкция запуска:
 - Откройте программу
 - Выберите нужну операцию, нажав на нужную цифру: Проверить пароль, Посомтреть историю и выйти из программы
 - Введите пароль и нажмите enter
 - программа вам выдаст оценку вашего пароля

## Примеры запуска:
запустил программу, выбрал цифров нужную мне операцию, если выбрал цифру 1, то нужно Ввести свой пароль и нажать Enter, вскоре программа выдаст оценку программы и снова предложит выбрать операцию, нажав на нужную цифру.
