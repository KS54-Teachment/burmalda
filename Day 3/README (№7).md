# TaskFlow

## Цель проекта
TaskFlow — это веб-приложение для управления задачами и командной работой. Цель проекта — помочь пользователям планировать задачи, отслеживать прогресс и эффективно распределять обязанности внутри команды.

## Описание команды
- **Хоменко Евгения** — руководитель проекта.
- **Черный Дамиан** — консультант.
- **Шигарев Егор** — технический специалист.
- **Чернышева Наталия** — документатор.
- **Губанов Артем** — специалист по безопасности. 

## Используемые технологии
- HTML5
- Git и GitHub
